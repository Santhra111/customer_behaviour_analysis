
# Customer Behaviour Analysis

## Overview
This project analyzes customer purchase behaviour to identify high-value customers, discount impact, and product performance.

## Tools Used
- Python (Pandas)
- PostgreSQL
- SQLAlchemy
- Power BI

## Business Questions Answered
- Who spends more than average?
- Do discount users still generate high revenue?
- Which products have the highest ratings?
- Which age group generates maximum revenue?

## Key Insights
- High-spending customers exist even among discount users.
- Certain age groups contribute disproportionately to revenue.
- Top-rated products are not always highest revenue generators.

## How to Run
1. Load dataset into PostgreSQL
2. Run SQL analysis queries
3. Build Power BI dashboard
